(1) 数值类型

  1. 整型：包括布尔值True, False。 八进制以0开头。十六进制以0x(X)开头；
  2. 长整型：无限大小的数（受限于虚拟内存），结尾添加l或L
  3. 浮点型：十进制或科学计数法（e或E表示10），如5.0e-2
  4. 复数：实数部分+虚数部分j/J，如7+3j
