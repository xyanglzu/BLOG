# 标签

```
新建标签: command + t
关闭标签: command + w
切换标签: command + 数字 / command + 左右方向键
切换全屏: command + enter
查找: command + f
```

# 分屏

```
垂直分屏: command + d
水平分屏: command + shift + d
```

# 命令

```
查看剪切板历史: command + shift + h
上一条命令: ctrl + p
搜索命令: ctrl + r
```



# 光标控制

```
清除当前行: ctrl + u
到行首: ctrl + a
到行尾: ctrl + e
前进后退: ctrl + f/b
删除当前光标的字符: ctrl + d
删除光标之前的字符: ctrl + h
删除光标之前的单词: ctrl + w
删除到文本末尾: ctrl + k
```

# 其他

```
清屏: command + r / ctrl + l / clear
```
